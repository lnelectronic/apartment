# คู่มือการใช้งาน — ระบบจัดการค่าเช่า App-Room

## ภาพรวมระบบ

ระบบแบ่งออกเป็น 3 หน้า เข้าถึงผ่าน URL เดียวกัน ต่างกันที่ parameter:

| หน้า | URL | ผู้ใช้ | หน้าที่ |
|------|-----|--------|---------|
| จดมิเตอร์ | `...exec?page=meter` | พนักงาน | จดมิเตอร์ ไม่เห็นยอดเงิน |
| จดมิเตอร์ (เต็ม) | `...exec?page=staff` | พนักงานอาวุโส | จดมิเตอร์ เห็นยอดเงิน |
| แอดมิน | `...exec?page=admin` | เจ้าของ/แอดมิน | Dashboard, บิล, จัดการห้อง |

---

## 1. หน้าจดมิเตอร์ (`?page=meter`)

เหมาะสำหรับพนักงานทั่วไป — จดมิเตอร์ได้อย่างเดียว **ไม่เห็นยอดเงินใดๆ**

### 1.1 หน้าตา

เมื่อเปิดหน้าจะเห็น 2 ส่วน:

**แถบห้องที่ยังไม่ได้จด** (ด้านบน)
- แสดงรายชื่อห้องที่ยังไม่ได้จดมิเตอร์เดือนนี้ เป็น chip สี
  - สีน้ำเงิน = ตึก C (ชยางกูร)
  - สีส้ม = ตึก R (Runway)
- ถ้าจดครบแล้วจะแสดง **✓ จดครบทุกห้องแล้ว**
- อัปเดตทันทีเมื่อบันทึกแต่ละห้องสำเร็จ

**ช่องค้นหาห้อง**
- กรอกเลขห้อง เช่น `C201` หรือ `R15`
- กด **ค้นหา** หรือ Enter

### 1.2 ข้อความ error ที่อาจพบ

| ข้อความ | ความหมาย | วิธีแก้ |
|---------|----------|---------|
| ไม่พบห้องนี้ กรุณาติดต่อแอดมิน | เลขห้องผิด | ตรวจสอบเลขห้องอีกครั้ง |
| ห้อง XX ว่างอยู่ ยังไม่มีผู้เช่า | ห้องนั้นยังไม่มีผู้เช่า | ไม่ต้องจด |
| ห้อง XX จดมิเตอร์เดือนนี้แล้ว ติดต่อแอดมินเพื่อแก้ไข | จดไปแล้ว | แจ้งแอดมินถ้าต้องการแก้ไข |

### 1.3 ขั้นตอนการจดมิเตอร์

1. กรอกเลขห้อง → กด **ค้นหา**
2. ระบบแสดง banner:
   - เลขห้อง + ชื่อผู้เช่า
   - สถานะ **ปกติ** (เขียว) หรือ **ค้างจ่าย** (แดง)
3. กรอก **มิเตอร์ไฟ "ถึง"** (ค่าเริ่มต้นระบบดึงมาให้อัตโนมัติ แก้ไขไม่ได้)
4. กรอก **มิเตอร์น้ำ "ถึง"** (เช่นเดียวกัน)
5. กด **บันทึก**
6. ระบบแสดง **บันทึกสำเร็จ!** พร้อมชื่อห้อง
7. กด **จดห้องถัดไป** เพื่อกลับไปจดห้องอื่น

> **หมายเหตุ:** ค่า "เริ่ม" คือเลขมิเตอร์สิ้นเดือนที่แล้ว ระบบดึงมาให้อัตโนมัติ ห้ามแก้ไข

## 2. หน้าจดมิเตอร์เต็มรูปแบบ (`?page=staff`)

เหมาะสำหรับพนักงานที่ต้องการเห็นยอดเงิน สามารถบันทึกรายการเพิ่มเติม และแก้ไขข้อมูลที่จดไปแล้วได้

### 2.1 ขั้นตอนปกติ (ยังไม่เคยจดเดือนนี้)

1. กรอกเลขห้อง → กด **ค้นหา**
2. ระบบแสดง banner:
   - เลขห้อง + ชื่อผู้เช่า
   - **ค้างเดือนก่อน** (ถ้ามี จะแสดงเป็นสีแดง) หรือ **ไม่มียอดค้าง** (เขียว)
3. กรอกมิเตอร์:
   - **ไฟ ถึง** — เลขมิเตอร์ไฟปัจจุบัน
   - **น้ำ ถึง** — เลขมิเตอร์น้ำปัจจุบัน
4. กรอก **รายการอื่นๆ** (ถ้ามี ไม่บังคับ สูงสุด 2 รายการ):
   - รายการที่ 1: ชื่อรายการ + จำนวนเงิน
   - รายการที่ 2: ชื่อรายการ + จำนวนเงิน
5. กด **คำนวณ**
6. ระบบแสดงหน้า **ยอดรวมประมาณการ**:
   - ค่าเช่า
   - ค่าเฟอร์นิเจอร์ (ถ้ามี)
   - ค่าไฟ (หน่วยที่ใช้ × ราคาต่อหน่วย)
   - ค่าน้ำ (หน่วยที่ใช้ × ราคาต่อหน่วย)
   - ค้างเดือนก่อน (ถ้ามี)
   - รายการอื่นๆ (ถ้ากรอก)
   - **รวมทั้งหมด**
7. ตรวจสอบยอด → กด **บันทึก**
   - ถ้าต้องการแก้ไข กด **แก้ไข** เพื่อกลับไปแก้
8. ระบบแสดง **บันทึกสำเร็จ!**

### 2.2 แก้ไขข้อมูลที่จดไปแล้ว

1. กรอกเลขห้องที่จดไปแล้ว → กด **ค้นหา**
2. ระบบแสดงข้อความ **"ห้องนี้มีข้อมูลเดือน X/YYYY แล้ว ต้องการแก้ไขหรือไม่?"**
3. กด **แก้ไขทับ** (ปุ่มแดง) เพื่อแก้ไข หรือ **ยกเลิก** เพื่อออก
4. ระบบโหลดข้อมูลเดิมมาให้แก้ไข
5. แก้ไขมิเตอร์/รายการ → กด **คำนวณ** → ตรวจสอบ → กด **บันทึก**

> **หมายเหตุ:** การแก้ไขทับจะอัปเดตเฉพาะมิเตอร์และรายการ **"จ่ายจริง"** และ **"สถานะ"** ที่แอดมินกรอกไว้จะไม่เปลี่ยนแปลง

## 3. หน้าแอดมิน (`?page=admin`)

### 3.1 เข้าสู่ระบบ

1. เปิดหน้า `?page=admin`
2. กรอก **รหัสผ่าน** → กด **เข้าสู่ระบบ** หรือ Enter
3. ถ้ารหัสผ่านถูกต้อง ระบบจะเข้าสู่ Dashboard ทันที
4. session เก็บไว้ใน browser — ปิดแท็บแล้วเปิดใหม่ไม่ต้อง login ซ้ำ (ตราบที่ยังเปิด browser อยู่)

**ออกจากระบบ:** กดปุ่ม **ออกจากระบบ** มุมบนขวา

---

### 3.2 แถบ Dashboard

แสดงสถิติภาพรวมของเดือนที่เลือก

**การเลือกเดือน**
- กด **◀ เดือนก่อน** / **เดือนถัดไป ▶** เพื่อเปลี่ยนเดือน
- กด **รีเฟรช** เพื่อโหลดข้อมูลใหม่

**การ์ดสถิติ**

| การ์ด | ความหมาย |
|-------|----------|
| ห้องทั้งหมด | จำนวนห้องในระบบ (46 ห้อง) |
| มีผู้เช่า | ห้องที่มีชื่อผู้เช่าอยู่ |
| ว่าง | ห้องที่ยังไม่มีผู้เช่า |
| จ่ายแล้ว | จำนวนห้องที่จ่ายครบเดือนนี้ |
| ยังไม่จ่าย | จำนวนห้องที่ยังค้างชำระ |
| ยอดที่ควรเก็บ | ผลรวมบิลทุกห้องเดือนนี้ |
| เก็บได้จริง | ยอดที่จ่ายแล้วจริง |
| ค้างชำระ | ยอดที่ยังไม่ได้รับ |

**รอการอ่านมิเตอร์**
- แสดงรายชื่อห้องที่มีผู้เช่าแต่ยังไม่ได้จดมิเตอร์เดือนนี้
- กดลิงก์ **อ่านมิเตอร์ ↗** (มุมขวาบน) เพื่อเปิดหน้าพนักงานในแท็บใหม่

**ตารางห้องค้างจ่าย**
- แสดงเฉพาะห้องที่สถานะ "ค้าง"
- คอลัมน์: ห้อง, ผู้เช่า, รวมเดือนนี้, ค้างก่อน, จ่ายแล้ว, ยังค้าง
- กด **ดูบิล** เพื่อดูใบแจ้งค่าเช่าของห้องนั้น

---

### 3.3 แถบจัดการบิล

บันทึกยอดรับเงินจริงและค่าปรับรายห้อง

**ขั้นตอนกรอกจ่ายจริง**

1. คลิกแถบ **จัดการบิล**
2. ตารางแสดงทุกห้องที่จดมิเตอร์แล้วเดือนนั้น
3. สำหรับแต่ละห้อง:
   - ช่อง **ค่าปรับ** — กรอกค่าปรับ (บาท) ถ้าไม่มีใส่ 0
   - ช่อง **จ่ายจริง** — กรอกจำนวนเงินที่ผู้เช่าจ่ายมาจริง
4. กด **บันทึก** ที่แถวนั้น
5. **สถานะ** จะเปลี่ยนอัตโนมัติ:
   - **จ่ายแล้ว** (เขียว) — เมื่อ จ่ายจริง ≥ รวม
   - **รอจ่าย** (แดง) — เมื่อ จ่ายจริง < รวม

> **หมายเหตุ:** กรอกทีละแถว แล้วกดบันทึกแต่ละแถว ไม่มีปุ่มบันทึกทั้งหมด

**ดูบิล**
- กด **ดูบิล** ที่แถวใดก็ได้ เพื่อดูใบแจ้งค่าเช่า

---

### 3.4 ใบแจ้งค่าเช่า (Bill Modal)

**วิธีดูบิล:** กด "ดูบิล" จากตารางค้างจ่าย (Dashboard) หรือตารางจัดการบิล

ใบแจ้งค่าเช่าแสดง:
- ห้อง + ชื่อผู้เช่า + เดือน
- ค่าเช่า
- ค่าเฟอร์นิเจอร์
- ค่าไฟ (มิเตอร์เริ่ม → ถึง / จำนวนหน่วยที่ใช้)
- ค่าน้ำ (มิเตอร์เริ่ม → ถึง / จำนวนหน่วยที่ใช้)
- รายการอื่นๆ (ถ้ามี)
- ค้างเดือนก่อน (ถ้ามี แสดงสีแดง)
- ค่าปรับ (ถ้ามี แสดงสีส้ม)
- **รวมทั้งหมด**
- สถานะ: **จ่ายแล้ว** / **รอจ่าย**

**พิมพ์บิล:** กด **พิมพ์** → เปิด dialog พิมพ์ของ browser
- หน้าจะพิมพ์เฉพาะใบแจ้งค่าเช่า ส่วนอื่นถูกซ่อนโดยอัตโนมัติ
- เหมาะสำหรับแชร์ทาง LINE (Screenshot จาก print preview)

**ปิด:** กด **ปิด** หรือกดพื้นที่นอก modal

---

### 3.5 แถบจัดการห้อง

แก้ไขข้อมูลพื้นฐานของห้อง และจัดการการย้ายออก

**ข้อมูลที่แก้ไขได้ต่อห้อง:**

| ช่อง | ความหมาย |
|------|----------|
| ชื่อ | ชื่อผู้เช่า (ว่าง = ห้องว่าง) |
| เช่า | ค่าเช่ารายเดือน (บาท) |
| เฟอร์ | ค่าเฟอร์นิเจอร์รายเดือน (บาท) |
| ไฟเริ่ม | เลขมิเตอร์ไฟเริ่มต้น ใช้เฉพาะเดือนแรก (fallback) |
| น้ำเริ่ม | เลขมิเตอร์น้ำเริ่มต้น ใช้เฉพาะเดือนแรก (fallback) |

**วิธีแก้ไข:**
1. คลิกแถบ **จัดการห้อง**
2. แก้ไขข้อมูลในช่องที่ต้องการ
3. กด **บันทึก** ที่แถวนั้น

> ห้องที่ไม่มีชื่อผู้เช่าจะแสดง badge **ว่าง** สีส้ม

**ย้ายผู้เช่าออก:**
1. กด **ย้ายออก** (ปุ่มแดง) ที่แถวห้องนั้น
2. ถ้ามียอดค้างชำระ ระบบจะแจ้งเตือนยอดค้างก่อน
3. ยืนยัน → ระบบล้างชื่อผู้เช่า, ค่าเช่า, ค่าเฟอร์นิเจอร์ออก
4. ห้องจะแสดงสถานะ "ว่าง"

> **ข้อมูลใน Sheet "บันทึก" ของเดือนที่ผ่านมาจะยังคงอยู่** — ย้ายออกไม่ได้ลบประวัติ

## 4. ลำดับการทำงานประจำเดือน

```
ต้นเดือน
  └─ พนักงานเปิด ?page=meter หรือ ?page=staff
       └─ จดมิเตอร์ทีละห้อง (ดูจากแถบ "ยังไม่ได้จด" ว่าเหลือห้องไหนบ้าง)

กลางเดือน / เมื่อผู้เช่าจ่ายเงิน
  └─ แอดมินเปิด ?page=admin → แถบ "จัดการบิล"
       └─ กรอก "จ่ายจริง" → กด บันทึก → สถานะเปลี่ยนเป็น "จ่ายแล้ว"

เมื่อต้องการส่งบิลให้ผู้เช่า
  └─ แอดมิน → ดูบิล → พิมพ์ / Screenshot แชร์ LINE

สิ้นเดือน
  └─ แอดมิน → Dashboard → ตรวจสอบยอดรวม ค้างชำระ ห้องที่ยังไม่จด
```

## 5. คำถามที่พบบ่อย

**Q: จดมิเตอร์ผิด จะแก้ได้ไหม?**
A: ใช้หน้า `?page=staff` → กรอกเลขห้อง → กด **แก้ไขทับ** แก้ได้เลย หรือแจ้งแอดมิน (หน้า meter.html แก้ไขไม่ได้)

**Q: ค่า "เริ่ม" มิเตอร์ดึงมาจากไหน?**
A: ดึงจากค่า "ถึง" ของเดือนก่อนหน้าอัตโนมัติ ถ้าไม่มีข้อมูลเดือนก่อน (เดือนแรก) จะใช้ค่า "ไฟเริ่มต้น / น้ำเริ่มต้น" ที่แอดมินตั้งไว้ในแถบจัดการห้อง

**Q: ค้างเดือนก่อนคำนวณอย่างไร?**
A: = รวมเดือนก่อน − จ่ายจริงเดือนก่อน (ถ้าจ่ายครบ = 0, ถ้าจ่ายไม่ครบหรือไม่จ่าย = ยอดที่เหลือ)

**Q: จ่ายเกินยอดได้ไหม?**
A: ได้ — ระบบจะบันทึกตามที่กรอก สถานะจะเป็น "จ่ายแล้ว" และเดือนหน้าค้างเดือนก่อนจะเป็น 0 (ไม่มีระบบคืนเงินส่วนต่าง)

**Q: ราคาน้ำ/ไฟต่อหน่วยเปลี่ยนได้ไหม?**
A: ได้ — แก้ไขได้โดยตรงใน Google Sheets Sheet "ตั้งค่า" row 2-3 (ต้องเปิด Spreadsheet เอง ไม่มีใน Admin UI)

**Q: เพิ่มห้องใหม่ได้ไหม?**
A: ห้องทั้งหมดถูกตั้งค่าตั้งแต่แรก (46 ห้อง) ถ้าต้องการเพิ่มต้องแก้ใน SheetSetup.gs และรัน `setupSheets()` ใหม่ (ต้องติดต่อผู้พัฒนา)

**Q: รหัสผ่านแอดมินลืม ทำอย่างไร?**
A: เปิด Apps Script Editor → Project Settings → Script Properties → ดู/เปลี่ยน `ADMIN_PASSWORDS`
