## ติดตั้งครั้งแรก (รันครั้งเดียว)

### `setupSheets()`
**เมื่อไหร่:** ตอนสร้าง Spreadsheet ใหม่ ยังไม่มีข้อมูลอะไรเลย  
**ทำอะไร:** สร้าง Sheet "ตั้งค่า", "บันทึก", "ประวัติผู้เช่า" พร้อม header และข้อมูลห้องทั้ง 47 ห้อง  
**⚠ คำเตือน:** ล้างข้อมูลทั้งหมดในทุก Sheet — ห้ามรันถ้ามีข้อมูลจริงอยู่แล้ว

---

### `safeSetupSheets()`
**เมื่อไหร่:** ต้องการ reset โครงสร้าง Sheet แต่มีข้อมูลผู้เช่าที่ต้องการเก็บไว้  
**ทำอะไร:** backup ข้อมูลห้อง (ชื่อ/ค่าเช่า) → รัน setupSheets() → restore ข้อมูลกลับ  
**ปลอดภัยกว่า** `setupSheets()` เพราะกู้ข้อมูลกลับได้

---

## Migration (รันครั้งเดียวบน Sheet ที่มีข้อมูลแล้ว)

### `migrateAddTenantFields()`
**เมื่อไหร่:** Sheet "ตั้งค่า" ยังมีแค่ 6 column (A-F) และยังไม่มี Sheet "ประวัติผู้เช่า"  
**ทำอะไร:** เพิ่ม header column G-J (วันย้ายเข้า, มัดจำ, เบอร์, Note) + สร้าง Sheet "ประวัติผู้เช่า"  
**ตรวจก่อนรัน:** เปิด Sheet "ตั้งค่า" → ดู cell G5 — ถ้าว่างอยู่ค่อยรัน  
**ป้องกันรันซ้ำ:** ตรวจ G5 อัตโนมัติ ถ้ามีข้อมูลแล้วจะข้าม

---

### `migrateAddWaterMin()`
**เมื่อไหร่:** Sheet "ตั้งค่า" ยังไม่มี column D (ราคาน้ำขั้นต่ำ)  
**ทำอะไร:** เพิ่ม column D "ราคาน้ำขั้นต่ำ (บาท)" และตั้งค่าเริ่มต้น 120 บาทให้ทั้งตึก C และ R  
**ตรวจก่อนรัน:** เปิด Sheet "ตั้งค่า" → ดู cell D1 — ถ้าว่างอยู่ค่อยรัน

## แก้ข้อมูลเก่า (one-time fix)

### `fixRecordSheetDates()`
**เมื่อไหร่:** Sheet "บันทึก" column A เก็บข้อมูลเป็น Date (ตัวเลข) แทนที่จะเป็น text เช่น "6/2026"  
**อาการ:** dashboard แสดงยอด 0, หรือบันทึกมิเตอร์ซ้ำ 2 แถว  
**ทำอะไร:** แปลง Date → string ทุก row ใน column A

## ทดสอบ / Debug

### `seedTestData()`
**เมื่อไหร่:** ต้องการข้อมูลปลอมสำหรับทดสอบ  
**ทำอะไร:** ใส่ชื่อผู้เช่า, ค่าเช่า, มิเตอร์ปลอมในหลายห้อง  
**⚠ ต้องลบออกก่อนส่งมอบ** ด้วย `clearTestData()`

### `clearTestData()`
**เมื่อไหร่:** ก่อนส่งมอบระบบ หรือหลังทดสอบเสร็จ  
**ทำอะไร:** ล้างข้อมูล Sheet "บันทึก" ให้เหลือแต่ header

### `listBackups()`
**เมื่อไหร่:** ต้องการดูว่ามี backup อะไรบ้าง  
**ทำอะไร:** แสดงรายชื่อ Sheet backup ใน Logger (ดูใน Execution log)

### `debugRecordSheet()`
**เมื่อไหร่:** สงสัยว่า column A ใน Sheet "บันทึก" เก็บเป็น Date หรือ string  
**ทำอะไร:** แสดง 5 แถวแรกใน Execution log พร้อม type ของ cell

## ข้อมูลสำคัญ

| Sheet | เก็บอะไร | Note column |
|-------|----------|-------------|
| ตั้งค่า | ข้อมูลห้อง + ผู้เช่า | Col J = Note ห้อง |
| บันทึก | มิเตอร์/บิลรายเดือน | Col A ต้องเป็น text เสมอ |
| ประวัติผู้เช่า | ข้อมูลผู้เช่าที่ย้ายออกแล้ว | append-only |
